# React + Vite

Small FrontEnd made with React and served on Vite.

## How to run the server:
### First time
1. ```npm install```
2. ```npm install react-bootstrap bootstrap```
3. ```npm install primereact```
4. ```npm run dev```

URL: http://localhost:5173/

### Other times
1. ```npm run dev```

## How to stop the server:
1. CTRL + C.
